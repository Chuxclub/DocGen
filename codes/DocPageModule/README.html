DocGen - DocPageModule

Auteur(s): Florian Legendre

Que contient ce dossier?

  • Header.m => classe abstraite avec que des méthodes statiques qui constitue une boîte à fonctions pour créer le header commun aux pages de manuel de DocGen (le bandeau orange avec les informations sur le projet et la rubrique 'Nous contacter')
  • Manual.m => classe abstraite ne contenant que des méthodes statiques qui constitue une boîte à fonctions pour créer les pages de manuel (aussi appelées 'notices') de DocGen. Une page de manuel, ou notice, étant définie comme étant un index + un README. Une page de manuel peut avoir une portée locale (ie. limitée au dossier courant et uniquement ce dossier) ou une portée globale (ie. récursive: le dossier courant et tous ses sous-dossiers.)
